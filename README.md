# 📝 オンライン原稿用紙 Pro (GIGA Edition)

日本の小・中学校の授業（GIGAスクール構想環境）での利用を想定して開発された、高機能かつ軽量なオンライン作文・原稿用紙アプリです。

Google Apps Script (GAS) をバックエンドとし、フロントエンドは React + Tailwind CSS を用いて **1つのHTMLファイル** にすべて集約されています。1学級40人が同時にアクセスしても落ちない堅牢な設計と、商用SaaSレベルの滑らかなUIUXを実現しています。

## ✨ 主な機能

### 👦👧 児童（生徒）向け機能

*   **リアルタイム原稿用紙プレビュー:** 入力した文章が即座に縦書きの原稿用紙フォーマット（20x20 または 15x16）に反映されます。
    
*   **賢い禁則処理:** 「追い出し」「ぶら下がり」の設定に対応。
    
*   **シームレスなオートセーブ:** タイピングが止まった瞬間にバックグラウンドで自動保存。児童の思考を妨げません。
    
*   **オフライン保護機構:** 学校のWi-Fiが切断されても、端末内に一時保存（キューイング）し、オンライン復帰時に自動で再送信します。
    
*   **先生とのコミュニケーション:** 提出機能や、返却された作文に対する先生からの赤ペン（添削）の確認機能。
    

### 👨‍🏫 教師向け機能

*   **一元管理ダッシュボード:** クラス全員の作文一覧、ステータス（下書き・提出済・完了・再提出）の管理。
    
*   **直感的な赤ペン添削:** 原稿用紙の文字を直接ドラッグして、滑らかにコメントを追加可能。
    
*   **🤖 AI自動添削機能:** Gemini API と連携し、児童の作文に対して「誤字脱字の指摘」「段落構成の助言」「温かいコメント」を自動生成します。
    
*   **楽観的UI (Optimistic UI):** ステータス変更が待ち時間0秒で画面に反映され、大量の採点作業もサクサク進みます。
    
*   **一括操作:** 複数人の作文の一括AI添削、一括完了、一括印刷に対応。
    

## 🛠 技術スタック

*   **フロントエンド:** React 18, Tailwind CSS, Babel (CDN経由で読み込み、ビルド不要)
    
*   **バックエンド:** Google Apps Script (GAS)
    
*   **データベース:** Google スプレッドシート (初回起動時に自動構築)
    
*   **外部API:** Google Gemini API (AI添削機能用)
    

## 🚀 究極のパフォーマンス最適化

このアプリは、限られたリソースのGASと学校のChromebook等でも快適に動作するよう、以下の最適化が施されています。

1.  **React.memoの極限活用:** 1文字入力するたびに400マス全体を再描画するのではなく、「変更があった行」のみをピンポイントで再描画します。
    
2.  **DOM直接操作による選択:** 教師のドラッグ添削時はReactのライフサイクルを回さず、Vanilla JSで直接DOMを操作することでネイティブアプリ並みの滑らかさを実現しています。
    
3.  **デバウンス処理:** オートセーブによるGASへのリクエスト集中を防ぎます。
    

## 📦 セットアップ方法

サーバーの契約や複雑な環境構築は一切不要です。Googleアカウントがあれば5分で使い始められます。

### 1. GASプロジェクトの作成

1.  [Google Apps Script](https://script.google.com/ "null") にアクセスし、「新しいプロジェクト」を作成します。
    
2.  プロジェクト名を「オンライン原稿用紙Pro」などに変更します。
    

### 2. コードの配置

1.  元からある `コード.gs` ファイルの中身を消去し、本リポジトリの `コード.gs` の内容を貼り付けます。
    
2.  左側の「＋」ボタンから「HTML」を選択し、ファイル名を `index` にして作成します。
    
3.  `index.html` の中身に、本リポジトリの `index.html` の内容をすべて貼り付けます。
    

### 3. デプロイ

1.  右上の「デプロイ」>「新しいデプロイ」をクリックします。
    
2.  種類の選択（歯車アイコン）から **「ウェブアプリ」** を選択します。
    
3.  設定を以下のようにします：
    
    *   説明: (任意)
        
    *   実行するユーザー: **自分 (あなたのアカウント)**
        
    *   アクセスできるユーザー: **全員**
        
4.  「デプロイ」をクリックし、承認を求められたらGoogleアカウントで許可を与えます。
    
5.  表示された **ウェブアプリのURL** にアクセスするとアプリが起動します！
    

> 💡 **Zero-Config DB構築** 初回起動時、バックグラウンドで自動的にあなたのGoogleドライブに「オンライン原稿用紙Pro_データ(React版)」というスプレッドシートが作成され、データベースとして機能し始めます。

## ⚙️ 初期設定（先生向け）

1.  アプリ画面右上の「先生用」ボタンをクリックします。
    
2.  初期パスワードは `admin` です。入力してログインします。
    
3.  右上の「歯車（設定）」アイコンをクリックし、「システム設定」を開きます。
    
    *   **パスワードの変更:** セキュリティのため、必ず変更してください。
        
    *   **Gemini APIキーの設定:** AI添削機能を使用する場合は、[Google AI Studio](https://aistudio.google.com/ "null") からAPIキーを取得して入力してください。
