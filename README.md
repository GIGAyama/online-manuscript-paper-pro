# 📝 オンライン原稿用紙 Pro (サーバーレス・P2P版)

「オンライン原稿用紙 Pro」は、GIGAスクール構想のタブレット端末に最適化された、ブラウザ上で動く原稿用紙＆作文指導アプリケーションです。

バックエンドサーバーやデータベース（GAS等）を一切使用せず、**PeerJSを用いたP2P通信**と**ブラウザのローカルストレージ**のみで動作する完全サーバーレス構成となっています。GitHub Pages等にHTMLファイルを1つ置くだけで、その日からすぐの授業で活用できます。

## ✨ 主な機能と特徴

### 🏫 授業に直結するリアルタイム通信（P2P）

*   **超カンタン接続:** 先生がルームを作成すると発行される「4桁のPINコード」を児童が入力するだけで、端末同士が直接繋がります。面倒なログインIDやパスワードは不要です。
    
*   **魔法の自動同期:** 児童が作文を書いている最中、定期的に先生の画面へ自動で同期されます。また、通信が切れたり日をまたいだりしても、次回接続時に自動で最新の添削データが児童に返却されます。
    

### ✍️ 児童向け最適化UI

*   **ふりがな（ルビ）完全対応:** 画面上の漢字には読みやすいレイアウトでふりがなが振られており、低学年の児童でも迷わず操作できます。
    
*   **レスポンシブデザイン:** タブレットの横画面・縦画面、スマートフォンの狭い画面でも、入力欄と原稿用紙プレビューが最適なレイアウトで表示されます。
    
*   **安心のローカル保存:** ネットワーク環境がない場合でも、端末内にデータを一時保存（オートセーブ）したり、ファイル（`.json`）として書き出して保存することが可能です。
    

### 👩‍🏫 プロ品質の教師用添削ツール

*   **直感的な赤ペン機能:** 原稿用紙のプレビュー画面上で、直したい文字をマウスや指でなぞるだけで、簡単に修正指示やコメントを追加できます。
    
*   **AI自動添削アシスト (Gemini API):** Googleの最新AIを活用し、誤字脱字のチェックや「児童の思考を促すアドバイス」をワンタップで生成します。（※個人情報保護のため、児童の名前はAIに送信されません）
    
*   **一括印刷・一括完了:** 選択した児童の作文だけをまとめてA4横の原稿用紙フォーマットで綺麗に印刷できます。
    

## 🚀 導入・デプロイ方法

本アプリは **HTML/CSS/JavaScript が全て統合された単一のファイル（`index.html`）** で構成されています。

1.  本リポジトリの `index.html` をダウンロード、またはフォークします。
    
2.  **GitHub Pages** やお好きなWebホスティングサービス（Vercel, Netlify, 学校内サーバーなど）に `index.html` をアップロードします。
    
3.  公開されたURLにアクセスするだけで、すぐに利用開始できます。
    

※ **AI添削機能を使用する場合:** 先生用画面の「設定（歯車マーク）」から、ご自身の **Gemini APIキー** を登録してください。APIキーは先生のブラウザ内のみに安全に保存されます。

## 📖 基本的な使い方

### 👩‍🏫 先生（ホスト）の操作

1.  アプリを開き、右上の「先生用」ボタンをクリックして先生モードに入ります。
    
2.  左側のパネルから **「通信ルームを開始」** ボタンを押します。
    
3.  画面に表示された **4桁のPINコード** を児童に伝えます。
    
4.  児童が接続し、作文を書き始めると、左側のリストに児童の名前と状況がリアルタイムで更新されていきます。
    
5.  リストから児童を選ぶと、右側に原稿用紙が表示されます。文字をドラッグして添削したり、右上の「AI添削」ボタンを活用して指導を行います。
    
6.  添削後、「返却（再提出）」または「完了」ボタンを押すと、即座に児童の画面に結果が反映されます。
    

### 🧒 児童（クライアント）の操作

1.  アプリを開き、画面上部の **「先生と接続」** ボタン（鎖のマーク）を押します。
    
2.  先生から教えられた **4桁のPINコード** を入力して接続します。
    
3.  題名、学年・クラス、名前を入力し、作文を書き始めます。
    
4.  下書きができたら **「提出」** ボタンを押して先生に送ります。
    
5.  先生から添削が返ってくると、画面右上の「通知（ベルのマーク）」が光ります。タップして先生のコメントを確認し、作文を直します。
    

## 🛠 技術スタック

*   **UI Framework:** React (18), Tailwind CSS
    
*   **P2P Communication:** PeerJS
    
*   **Text Diffing:** diff-match-patch
    
*   **AI Integration:** Google Gemini API (`gemini-2.5-flash`)
    
*   **Other Libraries:** SweetAlert2 (UI Alerts), Bootstrap Icons
    

## ⚠️ 注意事項・制限事項

*   完全サーバーレス（ブラウザのローカルストレージ依存）のため、**別の端末（別のブラウザ）で開くと、過去のデータは表示されません**。別の端末で続きを書きたい場合は、必ず「ファイル（.json）」として保存し、新しい端末で「開く」から読み込んでください。
    
*   先生のパソコンのブラウザ履歴やキャッシュを完全に消去すると、保存されていた児童の提出データも消えてしまいます。長期保存が必要な場合は、印刷機能やファイルダウンロード機能でバックアップを取るようにしてください。
